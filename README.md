## 📋 Descripción

Una plataforma de comercio electrónico completa y moderna desarrollada específicamente para Point of Sale (POS) en Honduras. Construida con Next.js, esta plataforma está diseñada para ofrecer una experiencia de compra fluida y atractiva, adaptada a las necesidades del mercado hondureño. La plataforma permite a los usuarios explorar un catálogo detallado de productos tecnológicos, realizar búsquedas específicas, gestionar un carrito de compras y completar el proceso de compra de manera intuitiva.

## ✨ Características Principales

### 🔍 Para Clientes

- **Exploración de Productos**: Navegación por categorías jerárquicas y búsqueda en tiempo real
- **Detalles de Producto**: Galería de imágenes, especificaciones técnicas, videos demostrativos
- **Gestión de Carrito**: Añadir/eliminar productos, actualizar cantidades, persistencia entre sesiones
- **Proceso de Compra**: Formulario de datos de envío y contacto con validación en tiempo real
- **Soporte al Cliente**: Formulario de contacto, preguntas frecuentes, solicitud de demostraciones
- **Experiencia Responsive**: Diseño adaptable a dispositivos móviles, tablets y escritorio
- **Localización**: Adaptado específicamente para el mercado hondureño

### ⚙️ Para Administradores

- **Panel de Administración**: Autenticación segura, gestión completa de productos (CRUD)
- **Gestión de Productos**: Añadir, editar, eliminar y organizar productos por categorías
- **Monitoreo de Solicitudes**: Visualización de pedidos, solicitudes de demostración y consultas
- **Reportes**: Estadísticas de ventas y comportamiento de usuarios

## 🛠️ Tecnologías Utilizadas

### Frontend
- **Next.js 14**: Framework React con renderizado del lado del servidor (SSR) y generación estática (SSG)
- **React 18**: Biblioteca para construir interfaces de usuario
- **Tailwind CSS**: Framework de utilidades CSS para un diseño rápido y consistente
- **TypeScript**: Superset de JavaScript que añade tipado estático
- **Zustand**: Biblioteca ligera para gestión de estado global
- **React Hook Form**: Para manejo eficiente de formularios con validación
- **Zod**: Biblioteca de validación de esquemas para TypeScript
- **shadcn/ui**: Componentes de UI accesibles y personalizables

### Backend
- **Next.js API Routes**: Endpoints serverless para funcionalidades del backend
- **Firebase Authentication**: Sistema de autenticación seguro para el panel de administración
- **Firestore Database**: Base de datos NoSQL para almacenamiento de productos
- **Supabase**: Base de datos PostgreSQL para almacenamiento de datos de clientes y solicitudes
- **Nodemailer**: Biblioteca para envío de correos electrónicos
- **Cloudinary**: Servicio de gestión de imágenes y videos
